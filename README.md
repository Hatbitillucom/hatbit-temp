# hatbit-temp

Temporary superproject for uncategorized HATBIT repositories.

These repos are parked here until they are properly categorized or archived.

## Submodules

| Submodule | Description |
|-----------|-------------|
| [issac](https://github.com/Hatbitillucom/issac) | issac |
| [Holonyak](https://github.com/Hatbitillucom/Holonyak) | Holonyak |

## Usage

```bash
git clone --recurse-submodules https://github.com/Hatbitillucom/hatbit-temp.git
git submodule update --init --recursive
```
